Address API Documentation

API สำหรับดึงข้อมูลที่อยู่ในประเทศไทย ได้แก่ ตำบล/แขวง อำเภอ/เขต จังหวัด รหัสไปรษณีย์
ผู้ใช้ทั่วไปสามารถเชื่อมต่อ API นี้เพื่อดึงข้อมูลไปใช้งานในระบบของตนเองได้ทันที

Base URL

http://localhost:3001

วิธีดึงข้อมูลที่อยู่ (ตำบล อำเภอ จังหวัด รหัสไปรษณีย์)

GET /api/address
  • province : กรองชื่อจังหวัด (เช่น กรุงเทพมหานคร)
  • city : กรองชื่ออำเภอ/เขต (เช่น ลาดพร้าว)
  • district : กรองชื่อตำบล/แขวง (เช่น จรเข้บัว)
  • postal : กรองรหัสไปรษณีย์ (เช่น 10230)
  • page : เลขหน้าที่ต้องการ (default 1)
  • limit : จำนวนข้อมูลต่อหน้า (default 50)
ตัวอย่างการเรียกใช้งาน
ดึงข้อมูลทุกตำบลในจังหวัดกรุงเทพมหานคร 
GET /api/address?province=กรุงเทพมหานคร
ดึงข้อมูลเฉพาะตำบล "จรเข้บัว" ในอำเภอ "ลาดพร้าว" 
GET /api/address?district=จรเข้บัว&city=ลาดพร้าว
ดึงข้อมูลด้วยรหัสไปรษณีย์ 
GET /api/address?postal=10230

หมายเหตุ
  • สามารถกรองได้พร้อมกันหลายเงื่อนไข เช่น ทั้งจังหวัดและอำเภอ
  • ข้อมูลที่ได้จะมีทั้ง ตำบล อำเภอ จังหวัด และรหัสไปรษณีย์ในแต่ละแถว
